## 🚩 Level: Bandit 0 &rightarrow; Bandit 1

### 🎯 Objective:
Connect to the server using SSH and retrieve the password for the next level from a file located in the home directory.

### 🔑 Concepts Learned:
- **SSH (Secure Shell):** A cryptographic network protocol for operating network services securely over an unsecured network.
- **Basic CLI Navigation:** Listing and reading simple files.

### 🛠️ Commands Explored:
* `ssh`: Used to connect to the remote machine. 
    * *Usage:* `ssh [username]@[host] -p [port]`
* `ls`: Lists directory contents.
* `cat`: Concatenates files and prints them to the standard output.
    * *Usage:* `cat [filename]`
